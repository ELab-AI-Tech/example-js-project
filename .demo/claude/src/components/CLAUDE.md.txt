# src/components conventions

These rules apply to every file under src/components/.

- Default export only. Use `export default function ComponentName`.
- Plain JSX — no TypeScript, no .tsx.
- Double quotes for all string literals.
- No icon library imports — none are installed in this repo.
- Props are destructured in the function signature: `function Card({ user })`, not `function Card(props)`.
- No CSS-in-JS. Apply class names from App.css or inline style only when the class does not exist yet.
